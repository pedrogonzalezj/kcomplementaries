Devo Coding Exercise - K Complementaries
==================================

### Requirements
+ java 8
+ maven 3 
### Big O time and Space Complexity
+ Time complexity is O(n) lineal where n is 2 times the input length
+ Space complexity is O(n) increases linearly by input size

### How to build the app
+ Go to app root directory and execute:
```
mvn clean package
```
+ Move to target directory and execute:
```
java -jar palindromes-1.0-SNAPSHOT.jar -k number
``` 
+ Introduce numbers separated by white spaces (or comma) to get K Complementaries
+ Enjoy!
